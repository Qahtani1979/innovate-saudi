# Strategy Components

## Page Components

### StrategyHub.jsx
Main dashboard for strategic planning.

### StrategicPlans.jsx
Listing page for all strategic plans.

### StrategicPlanDetail.jsx
Detail view with objectives, KPIs, and action plans.

## UI Components

### StrategicPlanCard
Displays plan summary.

### ObjectiveTree
Hierarchical objective visualization.

### KPIDashboard
KPI progress tracking dashboard.

### ActionPlanTimeline
Gantt-style timeline for action items.

### StrategyTemplateGallery
Browse and apply plan templates.

## Form Components

### PlanCreateWizard
Multi-step plan creation wizard.

### ObjectiveForm
SMART objective creation form.

### KPIConfigForm
KPI configuration with targets.

### ActionItemForm
Action item with dependencies.
